Configuration
-------------

- Just install the extension from TER.

- In the EM you can set:

  - the length of the password (5 – 20 chars)

  - the pattern the extension looks for. Just activate the checkbox and
    the extension checks if this kind of pattern will fit

  - the quantity of patterns to fit. If you activate all 4 patterns you
    can set that only 2 patterns have to fit, e.g. lowercase characters
    and digits or capitals and special characters. So you can specify your
    password conventions.

  - (> TYPO3 4.5) you can set a period of time. After not changing the
    user's password for this time, a popup after login will remind him.
