========================
Secure Backend Passwords
========================

:Extension key:
      be_secure_pw

:Version:
      x.x.x

:Language:
      en

:Description:
      This extension aims to harden the security of your TYPO3 CMS
      site by establishing secure password for backend users.

:Keywords:

This extension gives you the possibility to force secure passwords
from your BE users. You can setup the password patterns like e.g.
capitals, digits, etc. and a minimal length of the password.

After activating the extensions, your BE users must fit the patterns
and the length of the password to save a new one!
